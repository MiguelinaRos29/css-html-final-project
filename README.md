# DevCamp's Fantastic Fries 

Este proyecto es una réplica del sitio web **Fantastic Fries** realizado como práctica de HTML5 y CSS3.  
Incluye las secciones: **Home, About, Menu y Contact**.

##  Estructura
- `index.html` → Página principal  
- `about.html` → Información sobre la marca  
- `menu.html` → Catálogo de productos  
- `contact.html` → Formulario de contacto  
- `styles.css` → Hoja de estilos principal  

##  Tecnologías
- HTML5
- CSS3 (grid, flexbox y media queries)
- Google Fonts y CDN de Font Awesome

##  Cómo visualizarlo
1. Descarga la carpeta del proyecto.  
2. Abre `index.html` en el navegador o usa **Live Server** desde Visual Studio Code.
